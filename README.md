# Guess_the_number_game
A simple number guessing game built using Python's random module.

## Features
- Random number generation
- User input handling
- Hints (Higher/Lower)
- Tracks number of attempts

## Technologies Used
- Python 3
- random module

## How to Run

bash
python main.py


## Sample Output


Guess the number: 50
Guess Lower number please
Guess the number: 25
Guess higher number please
Guess the number: 37
You have guessed the number 37 correctly in 6 attempts.
